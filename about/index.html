二、学习历程

自从小学四年级家里买了第一台电脑，我每天放学的第一件事就是坐在电脑旁，从浏览网页、视频、玩游戏，到下载各种盗版游戏软件，让我感到乐此不疲，也因此对计算机产生了浓烈的好奇心。

记得高一那年去网吧用了在吾爱论坛上下载的一个网吧破解软件，开一次机玩一上午电脑，虽然还没用几次被网管发现教训了一顿，但小伙伴们羡慕的目光和那种成就感现在看来仍是回味无穷。可惜的是，后来因为学业原因，并未深入了解计算机。

高二的时候，免流、秒赞、还有各种卡盟都很火。我学习了基础的 html 语法，用免费主机搭建了一个秒赞网，虽然是下载的别人的源码，但当时也在班里传播了一阵，还是挺有趣的。

后来，我接触了 Adobe Premiere 视频剪辑软件。当时正在上高三，我报了艺术班，时间比普通高三学生多一些。B 站上出现了各种鬼畜视频，为此我学习了 Oeasy 的 Premiere 课程，想自己研究一下，这也是我大学中一个薄弱的收入来源。

到了大学，才开始接触编程语言，刚开始看了 C，后来看了 Python，都是断断续续，也没有做什么项目。又因为读的音乐专业，学习了 Cubase 等音乐相关软件。再后来到了大三，开始学习 Java，准备转行程序员。

三、未来计划

2022 年 9 月 1 日，学完了 Java 基础语法与 MySQL 数据库，准备继续学习 HTML + CSS + JavaScript。
